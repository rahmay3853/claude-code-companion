- Voice Tone: You can pick different vocal styles to change the personality of your assistant.
- Mute: Use the physical mute button in the app if you need to focus in silence for a while.

## 🔍 Troubleshooting common issues

If the application does not behave as expected, check these common items:

- If you hear no sound, check your Windows sound settings. Make sure the output device is correct. 
- If the app stops responding, close the program using the Task Manager and open it again.
- If the mic does not pick up your work, ensure you have given the app permission to access your microphone in Windows Privacy Settings.
- Keep your computer updated to the latest version of Windows for the best stability.
